## 🚀 Latest Presentation

### The Agentic Operator: Automating Adversarial Reasoning with Local AI
* **Conference:** International Conference on Emerging Trends in Cyber Security & Digital Protection (ICETCSDP 2026)
* **Organizer:** Department of Computer Science and Engineering, MSIT (Kolkata, India)
* **Date:** June 19-20, 2026
* **Artifacts:** [Presentation Hub](./Talks/2026/ICETCSDP/) 
* **Abstract:** 
  > The bottleneck in offensive operations is no longer tool execution speed—it is the cognitive human latency sitting within the "Think-Act-Observe" loop. This presentation explores the architectural shift from rigid, brittle automation scripts to adaptive, autonomous local AI agents capable of parsing live EDR telemetry, choosing alternative techniques, and navigating active directories at machine speed—all within an OpSec-safe, air-gapped environment.
